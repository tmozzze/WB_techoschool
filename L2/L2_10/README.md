# Go Sort Tool

Эта программа сортирует строки текста, аналогично утилите `sort` в Linux.

## Возможности

- Чтение строк из файла или стандартного ввода (STDIN)
- Сортировка по столбцу (`-k N`, разделитель по умолчанию — табуляция)
- Числовая сортировка (`-n`)
- Обратный порядок (`-r`)
- Только уникальные строки (`-u`)
- Сортировка по названию месяца (`-M`, например, Jan, Feb, …)
- Игнорирование пробелов в конце строки (`-b`)
- Проверка отсортированности (`-c`)
- Человеко-читаемая числовая сортировка (`-h`, например, 1K, 2M)

## Примеры использования

```bash
# Сортировка файла по числовому значению второго столбца
go run main.go -k 2 -n file.txt

# Сортировка файла, игнорируя пробелы в конце строк и оставляя только уникальные строки
go run main.go -b -u file.txt

# Проверка, отсортирован ли файл
go run main.go -c file.txt
````

## Как это работает

1. Разбор флагов командной строки с помощью `config.ParseFlags()`
2. Чтение строк из файла или STDIN
3. Разделение каждой строки на поля, если указан флаг `-k` (столбец)
4. Опциональное удаление дубликатов (`-u`) или проверка сортировки (`-c`)
5. Сортировка строк с учётом флагов:

   * `-n` → числовая сортировка
   * `-h` → человеко-читаемая числовая сортировка
   * `-M` → сортировка по месяцам
   * `-b` → игнорирование пробелов в конце строк
   * `-r` → обратный порядок
6. Вывод отсортированных строк в STDOUT

## Сводка флагов

| Флаг   | Описание                                                 |
| ------ | -------------------------------------------------------- |
| `-k N` | Сортировка по столбцу N (по умолчанию табуляция)         |
| `-n`   | Числовая сортировка                                      |
| `-r`   | Обратный порядок                                         |
| `-u`   | Только уникальные строки                                 |
| `-M`   | Сортировка по названию месяца (Jan, Feb, ...)            |
| `-b`   | Игнорирование пробелов в конце строк                     |
| `-c`   | Проверка отсортированности                               |
| `-h`   | Человеко-читаемая числовая сортировка (например, 1K, 2M) |
